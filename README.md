# EarthOnDoc

## 概述

EarthOnDoc类似于地球OL，但是没有游戏本体，它是一个思想实验开放平台。我们不提供“技能”“知识”“资源包”“MOD”本身，只提供介绍和列表，具体内容需要你自己在现实中寻找。我们没有“成就与进度”的奖励，也不记录哪些人达成（文明级别成就除外），奖励在哪里你自己去理会。

## 项目结构

EarthOnDoc/

┣ docs/ 文档

┃ ┣ profile/ 玩家信息，即个人信息

┃ ┣ perceptionsensory/ 感知系统和资源包

┃ ┣ progress/ 成就与进度系统

┃ ┣ study/ 学习系统，即技能和知识
